# Customer Segmentation & Personalized Marketing Engine

## 📌 Project Overview

Businesses often struggle to understand customer purchasing behavior, making it difficult to design effective marketing campaigns and improve customer retention.

This project builds an end-to-end customer analytics solution that segments customers based on their purchasing patterns and generates personalized marketing strategies using RFM Analysis and K-Means Clustering. An interactive Power BI dashboard provides actionable business insights for stakeholders.

---

## 🎯 Business Problem

Traditional marketing campaigns treat all customers equally, resulting in:

- Low campaign effectiveness
- Poor customer retention
- Inefficient marketing spend
- Difficulty identifying high-value customers

This project helps businesses understand customer behavior and implement targeted marketing strategies.

---

## 🎯 Objectives

- Analyze customer purchasing behavior
- Perform RFM (Recency, Frequency, Monetary) Analysis
- Segment customers using K-Means Clustering
- Identify high-value and at-risk customers
- Generate personalized marketing recommendations
- Build an interactive Power BI dashboard
